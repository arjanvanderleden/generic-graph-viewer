# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in Generic Graph View.

## File Operations

| Shortcut | Action |
|----------|--------|
| **Cmd/Ctrl + O** | Open file dialog |
| **Cmd/Ctrl + R** | Reload current document (or open most recent) |
| **Cmd/Ctrl + W** | Close window |

## Export

| Shortcut | Action |
|----------|--------|
| **Cmd/Ctrl + Shift + E** | Export as PNG |
| **Cmd/Ctrl + Shift + S** | Export as SVG |

## View

| Shortcut | Action |
|----------|--------|
| **Cmd/Ctrl + 0** | Zoom to fit all nodes |
| **Cmd/Ctrl + 1** | Zoom to selection |
| **Cmd/Ctrl + L** | Open Layout Settings |
| **Cmd/Ctrl + Shift + R** | Reload application (dev) |

## Selection

| Action | Method |
|--------|--------|
| Select node | Click on node |
| Toggle node in selection | Cmd/Ctrl + Click |
| Clear selection | Click on background |
| Select category | Click category in sidebar |
| Add category to selection | Shift + Click category |
| Remove category from selection | Cmd/Ctrl + Click category |

## Navigation

| Action | Method |
|--------|--------|
| Pan | Click and drag background |
| Zoom in | Scroll up / Pinch out |
| Zoom out | Scroll down / Pinch in |

## Standard Application Shortcuts

| Shortcut | Action |
|----------|--------|
| **Cmd/Ctrl + Q** | Quit application |
| **Cmd/Ctrl + M** | Minimize window |
| **Cmd/Ctrl + ,** | Preferences (macOS) |
| **Cmd/Ctrl + H** | Hide application (macOS) |
| **F11** | Toggle fullscreen |

## Editing (in search/inputs)

| Shortcut | Action |
|----------|--------|
| **Cmd/Ctrl + A** | Select all |
| **Cmd/Ctrl + C** | Copy |
| **Cmd/Ctrl + V** | Paste |
| **Cmd/Ctrl + X** | Cut |
| **Cmd/Ctrl + Z** | Undo |
| **Cmd/Ctrl + Shift + Z** | Redo |

## Platform Notes

### macOS
- Use **Cmd** key for shortcuts
- Native title bar integration
- Standard macOS menu bar

### Windows / Linux
- Use **Ctrl** key for shortcuts
- Some macOS-specific shortcuts may differ
- Menu bar is inside the application window
