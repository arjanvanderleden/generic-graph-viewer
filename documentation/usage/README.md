## Quick Overview

Generic Graph View allows you to:

- **Visualize** graph data from JSON files with automatic force-directed layout
- **Explore** nodes and their connections interactively
- **Filter** by categories to focus on specific parts of the graph
- **Search** for nodes by name, ID, or category
- **Select** single or multiple nodes to examine their properties
- **Export** your visualizations as PNG or SVG images
- **Customize** the force simulation parameters for optimal layouts

## Quick Start

1. Download the latest release for your platform
2. Install and launch the application
3. Open a JSON graph file (Cmd/Ctrl+O)
4. Explore your graph!
